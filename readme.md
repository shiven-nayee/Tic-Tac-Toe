## TIC TAC TOE

Tic Tac Toe, A classic game and quite possibly the oldest remedy for boredom 
out there that exists. Many people(myself included) spent many years trying to 
master this wonderful strategy game. Now you have the oppurtunity to do so as
well! Master the skills of tic tac toe today!

![Starting game](https://git.generalassemb.ly/raw/shiven/js-tictactoe-homework/master/readme-assets/start.png?token=AAAb62AJBqeokPjqkWZYMvU2mXR_XBDpks5Zk1VewA%3D%3D)

The game begins with a 3 by 3 board where 2 players can place their respective
game pieces. 

To win the game you just need to get 3 of the same pieces in a row.

![Won game](https://git.generalassemb.ly/shiven/js-tictactoe-homework/blob/master/readme-assets/win.png?raw=true)
